websocket - WebSocket 13 support

Message

A subclass of bytes that encodes a single message.

is_ping

Should never appear if you’re using a Reader (it handles pings automatically.)

is_pong

Whether this message is a response to a previously sent ping.

is_data

Whether this message is some binary data.

is_text

Whether this message is text (in UTF-8, I think?)

is_close

Whether this message is the last one sent over this channel. The connection has already been closed.

Reader

A special coroutine-like object that waits for the next message and returns it when used with yield from. Raises asyncio.CancelledError if the connection gets closed; the first argument of the exception, if it exists, is a Message received before closing the connection (it will have is_close set to True.)

Writer

An object that allows one to send messages easily.

ping payload

Send a ping.

pong payload

Respond to a ping.

text payload

Send some text. Note that data must still be a bytes object.

data payload

Send some binary data.

close code data

Send a close message. No other messages can be sent after that.
